## maven deploy 方法
```shell
mvn deploy -DaltDeploymentRepository=${distribute_id}::default::file:D:\GitHub
```

## maven 升级版本号
```shell
// 升级
mvn versions:set -DnewVersion=1.0.1-SNAPSHOT
// 回退
mvn versions:revert
// 确认
mvn 
```
